- {guilabel}`TYPE`: Distortion type

- {guilabel}`LPF IN`: Pre-distortion Low-pass filter
- {guilabel}`LPF OUT`: Post-distortion Low-pass filter
- {guilabel}`IN`: Stereo audio input
- {guilabel}`OUT`: Stereo audio output
